## Шкала оценки рисков

### Уровень риска (Risk = L × I)

- **1-5** - Низкий (Low) - Принять
- **6-10** - Средний (Medium) - Снизить/Принять
- **11-15** - Высокий (High) - Снизить
- **16-25** - Критический (Critical) - Немедленно снизить

## Quick Wins

**Высокий ROI, быстрая реализация:**

1. **R7** - Валидация входных данных (Risk: 8) - Уже частично реализовано, требует доработки
2. **R1** - Rate limiting для /login (Risk: 12) - Критично для безопасности
3. **R11** - Сканирование уязвимостей (Risk: 12) - Интеграция в CI/CD
